# MRD-1X YAML Schema (ES/EN)

## ES — Convenciones
- Fuera de Ω_I sólo entran **parámetros finitos + error**, nunca funciones libres.
- El runner registra `input_sha256`, `env_fingerprint`, `versions`, `seed`.

## EN — Conventions
- Outside Ω_I only **finite parameters + error** are allowed, never free functions.
- Runner records `input_sha256`, `env_fingerprint`, `versions`, `seed`.
