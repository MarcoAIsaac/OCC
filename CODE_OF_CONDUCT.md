# Code of Conduct

Este proyecto adopta un estándar de conducta profesional para mantener un entorno seguro y colaborativo.

## Principios

- Respeto y trato digno.
- Críticas al trabajo, no a las personas.
- Colaboración enfocada en reproducibilidad, claridad y auditoría.

## Reporte

Si necesitas reportar un comportamiento inapropiado, abre un issue marcado como **private/security** (si aplica) o contacta al mantenedor a través de GitHub.
